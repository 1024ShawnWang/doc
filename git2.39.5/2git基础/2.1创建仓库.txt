
#1.有两种方式获得一个git仓库    

#1.1 将尚未进行版本控制的本地目录，转为git仓库 #自己造一个船新版本
    1.1.1 进入该目录，使用 git init命令
    1.1.2 目录多一个.git文件，但是目录中的文件并没有被跟踪
    1.1.3 git add  (标记为已经暂存
    1.1.4 git commit (提交


#1.2 从现有的git仓库，克隆一个                 #从别人照抄     （好像做作业)
    1.2.1 $ git clone https://github.com/libgit2/libgit2  执行此命令的目录下面对多一个子目录libgit ，里面有所有你想要的东西了
    1.2.2 $ git clone https://github.com/libgit2/libgit2 mylibgit 使用这个方式可以 自定义 文件夹名称
    1.2.3 git支持多种协议，这里是https协议，也可以是使用git或者ssh协议
